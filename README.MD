# How to use
You are recommend to delete file [switch/tinfoil/locations.conf] before adding server

It will auto add and update all of the alternative shops.

```
Protocol: https
Host: raw.githubusercontent.com
Path: bad1dea/tinfoil_json/main/tinfoil.json
Titles: khuong
```

NOTES: It will take around 3-5 minutes to load the server.
